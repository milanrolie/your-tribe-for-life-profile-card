## Installatie

1. Download de code als zip bestand en open deze map in je Code editor.
2. Voer de volgende commands uit in je terminal binnen het project: <br>
   `npm install`<br>
   `npm install gsap`<br>
   `npx @slicemachine/init@latest`<br>
   `npm i @studio-freight/lenis`<br>
3. Enjoy!
